RSS媒体阅读器 V2.0
项目简介
纯前端离线 RSS 阅读器，无需后端，仅单 HTML 文件运行，支持标准 RSS/Atom 订阅源解析，适配手机 / 电脑双端，内置多代理自动轮换解决跨域问题，收藏、批量导入导出、媒体图片 / 视频预览下载一体化工具。
在线直达演示（一键复制链接）：
https://hanyee2023.github.io/rss-twitter-viewer/

使用教程
方式 1：在线直接使用（推荐，无需本地文件）
复制直达链接，任意浏览器打开：
https://hanyee2023.github.io/rss-twitter-viewer/
切换「添加订阅」，输入自定义名称、分类、RSS 链接，点击确认添加；
返回主页自动加载全部订阅内容；
支持滚动分页加载、图片全屏预览、视频播放、收藏 / 下载媒体。
方式 2 本地电脑调试
下载仓库内 index.html 单文件；
终端执行 python -m http.server 启动本地静态服务；
浏览器访问本地地址调试，避免 file:// 协议跨域拦截。
数据备份迁移
管理订阅页 → 一键导出，自动下载 JSON 备份文件；
新设备打开项目 → 一键导入，选择备份文件即可恢复全部订阅。
部署教程（自己搭建 GitHub Pages）
新建 GitHub 仓库，上传 index.html 文件；
仓库设置 → Pages，开启静态站点；
生成专属在线访问链接，和本项目使用方式完全一致。
注意事项
公共代理存在限流、偶尔失效，失效后等待片刻或更换 RSS 源；
微信内置浏览器跨域限制严重，建议复制链接到系统浏览器打开；
所有订阅、收藏数据保存在浏览器本地，清除浏览器数据会丢失备份，建议定期导出 JSON；
仅支持标准 RSS/Atom 格式链接，非标准源会出现加载空白。
